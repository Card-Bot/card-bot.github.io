API DOCUMENTATION




This is the API documentation for CardBot

The base for all GET requests is htttps://card-bot.github.io/api


Searching

GET /cards/{term}.json or /cards/all.json

Search parameters:

  • Space - _
  • _ - __
  • ? - ~q
  • ! - ~e
  • ' - ~a
  • " - ~q

Sending a GET request will return the result, or "error" if nothing is found


GET /:info.json

Use this to get info, such as current version, author, license, etc


GET /cards/set/{set}.json

Sending a GET request will return all the cards and data for the set. Organized by id (0, 1, etc)
